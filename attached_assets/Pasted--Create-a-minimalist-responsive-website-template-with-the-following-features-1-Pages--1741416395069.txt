**"Create a minimalist, responsive website template with the following features:**  
1. **Pages**: `Home`, `Blog`, `Portfolio`, `About` (all linked via a clean navigation bar).  
2. **Design**:  
   - Inspired by *Future Imperfect* (HTML5 UP): Grid layout, featured post cards, subtle animations, and a clean sidebar (optional).  
   - Modern typography (Google Fonts: *Inter* or *Roboto*).  
   - Dark/Light mode toggle.  
   - Responsive for mobile/desktop.  
3. **Blog Section**:  
   - Daily post layout with dates, tags, and categories (e.g., *Python*, *Data Science*).  
   - Auto-generated blog post pages from Markdown files (or JSON data).  
   - Search bar and archive section.  
4. **Portfolio**:  
   - Grid layout with project cards (title, image, short description, GitHub/Kaggle links).  
   - Filterable by tags (e.g., *Machine Learning*, *Visualization*).  
5. **SEO Optimizations**:  
   - Meta tags (OG, Twitter cards) in `<head>`.  
   - Semantic HTML (header, article, section, footer).  
   - Dynamic page titles, alt text for images, and auto-generated sitemap.xml.  
   - Clean URLs (e.g., `/blog/post-title`).  
6. **Tech Stack**:  
   - Static HTML/CSS/JS (no backend).  
   - Optional: Integrate a lightweight static site generator (e.g., *11ty* or *Jekyll*).  
7. **Extras**:  
   - Social media icons (GitHub, LinkedIn, Twitter) in footer.  
   - RSS feed for blog posts.  
   - Loading optimizations (lazy-load images, minified CSS/JS).  

**Deliverables**:  
- A `index.html`, `blog.html`, `portfolio.html`, `about.html`.  
- Separate CSS/JS folders with `styles.css` and `scripts.js`.  
- A `posts/` folder with sample Markdown blog posts.  
- A `config.yml` file for site-wide settings (title, author, social links).  
- A README.md with deployment instructions to GitHub Pages.  

**Style Reference**:  
- Use the *Future Imperfect* theme’s card-based layout and hover effects.  
- Prioritize readability with line spacing (1.6rem+) and contrast ratios.  
- Include a *Featured Post* section on the home page (like a hero banner)."**
